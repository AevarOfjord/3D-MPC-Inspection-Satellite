# Code of Conduct

This project expects professional, respectful collaboration.

## Expected Behavior

- Be respectful and constructive.
- Focus feedback on code and behavior, not people.
- Assume good intent and ask clarifying questions before escalating.

## Unacceptable Behavior

- Harassment, discrimination, or personal attacks.
- Repeated disruptive behavior after warning.
- Publishing private information without consent.

## Enforcement

Maintainers may edit, lock, or remove content that violates this policy and may restrict project access when necessary.
